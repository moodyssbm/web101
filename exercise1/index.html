So we're just getting started learning html & css. It's essential that you understand how the syntax works so that's why we're gradually easing you in to all this "code". The best advice is to take advantage of the resources available to you to learn more. Here are a few suggestions to helping you learn to improve your skills with CSS layouts:

  1. Books. There are many great books published about CSS. Some of my favorites are: The Zen of CSS Design, Designing with Web Standards, CSS Mastery, CSS Hacks & Filters, and CSS - The Definitive guide.
  2. Online Resources. Here are just a few site's to help you in your journey to learning and mastering CSS:
    • A List Apart (http://www.alistapart.com) A digital magazine for web designers & developers
    • CSS Beauty (http://www.cssbeauty.com) A showcase of CSS sites, also archiving latest CSS techniques and articles
    • CSS Zen Garden (http://www.csszengarden.com) Take your stab at creating the "zen" of all sites.
    • CSS-Tricks (https://css-tricks.com) Constantly updated with new CSS tutorials, tips, tricks and techniques.
    • Web Designer Depot (http://www.webdesignerdepot.com/2009/08/250-resources-to-help-you-become-a-css-expert/) A huge list of CSS resources
    • CSS Basics (http://www.cssbasics.com) Beginning CSS? Here's a good starting place.
    • W3 Schools (http://www.w3schools.com/css) Another great place for beginners.
  3. Forums. Many of these same sites have forums for you to post questions. Often times CSS gurus monitor the sites regularly and can answer questions regularly.
  4. Learn from the CSS superheroes. That's right, there are a handful of designers who are well respected in the industry and visiting their site will inspire you to perhaps be as great as some of these "superheroes". Here's a nice presentation on "How to become a web design superhero"
  5. Learn by doing. Seriously, you can read, reference, and discuss CSS layouts all you like but actually getting hands on experience is the best way to learn them. Over time you'll gain experience as well as confidence.
